Object-Oriented Programming Concepts - Self-Study

üìÖ October 2025 ‚è±Ô∏è 5 hours üìö Self-Study

Activity Type: Self-directed learning and practice

What - Why is this activity important?

Object-oriented programming (OOP) is a fundamental paradigm used in many modern programming languages. Understanding OOP principles like encapsulation, inheritance, and polymorphism is essential for designing scalable, maintainable software systems and is critical knowledge for any software developer.

So What - What happened during the activity?

I studied OOP concepts through online tutorials and textbooks, learning about classes, objects, methods, and the principles of abstraction, encapsulation, inheritance, and polymorphism. I practiced implementing these concepts by creating small programs that modeled real-world scenarios using classes and objects.

What did it mean to me?

Understanding OOP changed how I think about program structure. Instead of writing procedural code, I learned to model problems using objects and their relationships. This approach makes code more organized, reusable, and easier to maintain.

How has it helped you develop?

I applied OOP principles extensively in the NCT Bus Ticket System project, creating four well-designed classes with clear responsibilities: Ticket, TicketManager, Purchase, and PurchaseManager. This modular design made the code easier to understand, test, and maintain. The separation of concerns principle I learned through OOP study resulted in cleaner, more professional code.

Now What - What are your next steps?

I want to deepen my understanding of design patterns and how they apply OOP principles to solve common programming challenges. Learning about SOLID principles and practicing their application in larger projects will help me write even more maintainable code. I also want to explore how OOP concepts apply in different programming languages like Java and C++.
