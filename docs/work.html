NCT Bus Ticket Purchase System

Python | Object-Oriented Programming | File Handling

Project Overview

Module: COMP10121 - Foundations of Computer Programming

Date: January 2026

Technologies: Python 3.11, CSV processing, Git version control

Status: Completed with 100% test pass rate

Description

Developed a comprehensive command-line Bus Ticket Purchase System for Nottingham City Transport that processes real ticket data from CSV files. The system features dual interfaces: a customer portal for browsing and purchasing tickets, and an administrative dashboard for monitoring prices and viewing system statistics.

The application demonstrates core programming concepts including object-oriented design, data structures, file handling, and user input validation. It successfully manages 104 ticket records across different categories and passenger classes.

Key Features

  • Customer Interface: Browse tickets by category, view detailed information, purchase tickets with quantity selection, and receive transaction receipts
  • Administrator Interface: Update ticket prices with optional CSV persistence, view system statistics including total tickets and categories, and monitor purchase records
  • Data Management: CSV file processing using Python's DictReader, persistent storage of purchase records, and efficient data structures (lists, dictionaries, sets)
  • Error Handling: Comprehensive try-except blocks for file operations, input validation with range and type checking, and graceful degradation under error conditions
  • Version Control: Systematic progress tracking using Git on NTU's Olympus GitHub platform

Technical Implementation

Object-Oriented Design: Implemented four main classes with clear separation of concerns:

  • Ticket Class: Data representation for individual tickets with price formatting methods
  • TicketManager Class: Collection management for all tickets with search and filter capabilities
  • Purchase Class: Transaction data modeling with receipt generation
  • PurchaseManager Class: Handles persistence operations for purchase records

Testing Strategy: Developed comprehensive testing methodology including:

  • Unit testing for individual methods and functions
  • Integration testing for workflow sequences
  • Edge case testing for boundary conditions and error scenarios
  • User acceptance testing with various input patterns
  • Result: 15/15 tests passed (100% success rate)

Challenges & Solutions

Challenge: Ensuring accurate price conversions between pence (CSV storage) and pounds (user display)

Solution: Implemented consistent conversion methods in the Ticket class with thorough validation and testing

Challenge: Managing data persistence across program sessions

Solution: Developed file append operations for purchase records and optional CSV updates for price changes, ensuring data integrity

Challenge: Creating robust input validation to prevent user errors

Solution: Implemented comprehensive validation system with range checking, type validation, and confirmation prompts for critical actions

Learning Outcomes

This project significantly enhanced my understanding of:

  • Object-oriented programming principles and class design
  • File processing techniques and CSV data manipulation
  • Error handling strategies and defensive programming
  • Software testing methodologies and quality assurance
  • Version control practices using Git
  • Professional code organization and documentation

Evidence: Complete source code, design documentation, and test results available in project repository

Documentation: 15-page technical report including use case analysis, block flow diagrams, class specifications, testing strategy, and critical evaluation

Impacts of Digitalization on SDG 13: Climate Action

Research | Academic Writing | Sustainability

Project Overview

Module: COMP10111 - Academic and Professional Development

Date: December 2025

Type: Academic Research Report

Word Count: 1,486 words

Description

Conducted comprehensive research examining how digitalization and Information and Communication Technologies (ICTs) impact progress toward Sustainable Development Goal 13: Climate Action. The research analyzed the ethical, social, legal, and environmental implications of technology deployment in addressing climate change.

This project demonstrates research skills, critical analysis, academic writing, and the ability to synthesize complex information from multiple peer-reviewed sources into a coherent argument.

Research Focus Areas

  • Ethical Impact: Examined data privacy concerns and ethical conflicts in technology transfer between developed and developing nations
  • Social Impact: Analyzed the digital divide, equitable access to technology, and citizen participation in smart city initiatives
  • Legal Impact: Investigated regulatory frameworks and their role in driving green technology innovation through compliance and incentives
  • Environmental Impact: Explored both mitigation strategies (emission reduction through optimization) and adaptation strategies (building urban resilience)

Key Findings

Mitigation through Technology:

  • R&D investment in the chemical sector shows significant GHG emission reduction
  • Digital Decision Support Frameworks can achieve 35% carbon footprint reduction in industrial processes
  • Technology integration enables effective control of the water-energy nexus

Adaptation through Digital Systems:

  • Early Warning Systems (EWS) use IoT and AI for predicting climate-induced risks
  • Digital twin strategies enhance flood resilience in smart cities
  • Real-time monitoring enables proactive climate action responses

Critical Challenges:

  • Digital divide threatens equitable access in developing countries
  • Data privacy and security concerns in extensive data collection
  • Risk of techno-centric solutions reinforcing existing social inequalities

Research Methodology

  • Reviewed 9 peer-reviewed academic sources from sustainability and technology journals
  • Analyzed case studies from Brazil, Germany, and other international contexts
  • Examined quantitative data on emission reductions and technology impacts
  • Synthesized findings across ethical, social, legal, and environmental dimensions
  • Applied proper academic referencing using Harvard style

Skills Demonstrated

  • Academic research and literature review techniques
  • Critical analysis of complex technical and social issues
  • Synthesis of information from multiple sources
  • Professional academic writing with proper structure and referencing
  • Understanding of technology's role in addressing global challenges
  • Evaluation of trade-offs between technological solutions and social equity

Evidence: Complete research report with figures, tables, and comprehensive reference list

Format: Professional academic report following university guidelines
