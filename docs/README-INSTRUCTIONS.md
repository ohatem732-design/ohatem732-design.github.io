## üÜò Troubleshooting

**Website not showing?**
- Wait 2-3 minutes after uploading
- Check Settings ‚Üí Pages shows "/docs" folder selected
- Make sure files are IN the /docs folder, not in root

**Images not loading?**
- Make sure image filename matches exactly (case-sensitive)
- Check the image is in the same folder as the HTML files

**Need to edit something?**
- Click on the file in GitHub
- Click the pencil icon (edit)
- Make your changes
- Click "Commit changes"
- Wait 1 minute, then refresh your website
